## 🌟 Descripción

**YT-DLP GUI** es una aplicación diseñada para simplificar al máximo el uso de la potente herramienta de línea de comandos `yt-dlp`. Construida con **CustomTkinter**, ofrece una experiencia de usuario moderna, fluida y con un aspecto oscuro premium, permitiéndote descargar contenido multimedia de miles de sitios web sin tener que tocar la terminal.

## ✨ Características Principales

*   🎨 **Interfaz UI/UX Premium**: Diseño oscuro y moderno pensado para el confort visual.
*   ⚡ **Múltiples Formatos y Calidades**: Descarga la mejor calidad pre-fusionada (1080p, 720p, 480p) o extrae el audio directamente a MP3 / M4A.
*   ✏️ **Edición de Metadatos**: Renombra el video o audio de destino antes de descargarlo con una amplia área de edición.
*   ⚙️ **Integración con FFmpeg**: Soporte completo para combinar pistas de audio y video de altísima calidad.
*   📊 **Monitor de Progreso Avanzado**: Barra de progreso en tiempo real que muestra el porcentaje, velocidad de descarga y tiempo estimado (ETA).
*   🛠️ **Consola de Depuración Integrada**: Si algo falla, despliega la consola inferior para ver el registro completo y solucionar problemas al instante.
*   🔗 **Soporte Multired**: Integración directa con las redes de *ServicePCGlew* y botones dedicados para *Acerca de* y soporte al creador (*Matecito*).

## 🚀 Instalación y Uso

1.  Descarga el archivo compilado `Yt-Dlp-GUI.exe` desde la sección de [Releases](https://github.com/servicepcglew/Yt-Dlp-GUI/releases).
2.  Asegúrate de contar con `yt-dlp.exe` y `ffmpeg.exe` en tu sistema (puedes seleccionarlos desde la propia aplicación).
3.  Pega tu enlace, selecciona la calidad deseada, ajusta el título y haz clic en **Descargar Video**.
